# Guide to Cell Painting for Biologists

So you’re a biologist interested in Cell Painting and don’t know where to get started? We’ve got you covered! For each question below, we’ve arranged the resources in increasing order of detail and/or specificity, so you may find that you don’t need to interact with every resource to glean the information you need.

## What is Cell Painting?

- Start by watching AskErinDearBeth Episode 23 - [Introduction to the Cell Painting Assay](https://www.youtube.com/watch?v=9-d6DoHtgUY) for a casual podcast introduction.  
- Read the introduction in the most recent [Cell Painting Protocol paper](http://doi.org/10.1038/s41596-023-00840-9)

## What can Cell Painting do?

- Read the recent review [Cell Painting: a decade of discovery and innovation in cellular imaging](https://www.nature.com/articles/s41592-024-02528-8)  
- Read a prior review [Image-based profiling for drug discovery: due for a machine-learning upgrade?](https://doi.org/10.1038/s41573-020-00117-w)  
- Check out talks on Youtube playlists from Anne Carpenter ([https://broad.io/CarpenterVideos](https://broad.io/CarpenterVideos)) and Shantanu Singh ([https://broad.io/SinghVideos](https://broad.io/SinghVideos))

## I’d like to modify the Cell Painting Assay. Can I do that?

You sure can! The Cell Painting Assay is very robust to changes.

- Start by watching Ask Erin Dear Beth Episode Episode 30: [Modifying the Cell Painting Assay](https://youtu.be/0PNmjJrmbG8) for a casual podcast introduction.  
- Read the most recent [Cell Painting Protocol paper](http://doi.org/10.1038/s41596-023-00840-9) for quantification of how modifications affect the strength of the assay and [Assessing the performance of the Cell Painting assay across different imaging systems](https://doi.org/10.1002/cyto.a.24786) for further modifications related to the microscopes themselves.  
- Look in the [Cell Painting wiki](https://github.com/carpenter-singh-lab/2023_Cimini_NatureProtocols/wiki) for some of the modifications that were tested in the development of the Cell Painting Assay.  
- The Cell Painting Gallery hosts a LOT of Cell Painting data. You can find many examples of published datasets using a variety of modified Cell Painting protocols [in the Cell Painting Gallery](https://github.com/broadinstitute/cellpainting-gallery).

## How do I perform my own image analysis on a batch of Cell Painting images?

- If you don’t already have a strong background in quantitative bioimaging, read the [Bioimaging Guide](https://www.bioimagingguide.org/) and its [companion publication](https://journals.plos.org/plosbiology/article?id=10.1371/journal.pbio.3002167) for background information. Watch Ask Erin Dear Beth Episode 20 - [Intro to the Bioimaging Guide](https://youtu.be/Upqfn9ALXz8) for a podcast introduction.  
- The most recent [Cell Painting Protocol paper](http://doi.org/10.1038/s41596-023-00840-9) includes the image analysis workflow.  
- The [Image Based Profiling Handbook](https://cytomining.github.io/profiling-handbook/01-overview.html) is a step-by-step protocol using a specific tech stack ([Distributed-CellProfiler](https://github.com/CellProfiler/Distributed-CellProfiler) on AWS on images generated by a PerkinElmer microscope, then collating the data with [cytominer-database](https://github.com/cytomining/cytominer-database) and analyzing it with [pycytominer](https://github.com/cytomining/pycytominer)) but the basic procedure for running a Cell Painting experiment is the same no matter the microscope or the processing platform.  
- An assortment of Cell Painting pipelines for CellProfiler (tuned for different cell types and microscope parameters) are available in [this repository](https://github.com/broadinstitute/imaging-platform-pipelines). The current standard pipelines on which we build all our pipelines currently are in [JUMP Production pipelines](https://github.com/broadinstitute/imaging-platform-pipelines/tree/master/JUMP_production).  
- Watch a [video walkthrough of the JUMP Production CellProfiler analysis pipeline](https://www.youtube.com/watch?v=Ow9sqGqoqXw).  
- Watch Ask Erin Dear Beth Episode 3 - [QC on High Content Imaging Datasets using CellProfiler and Fiji](https://www.youtube.com/watch?v=5pAcdd_Nn00) for details on our QC step in the Cell Painting Assay  
- Watch Ask Erin Dear Beth Episode 14 - [Illumination Correction in CellProfiler (Two Ways!)](https://youtu.be/4CKl-6TAAS4) for details on the illumination correction we perform in the Cell Painting Assay  
- Read the blog post [When to Say Good Enough](https://carpenter-singh-lab.broadinstitute.org/blog/when-to-say-good-enough) to decide when to stop trying to tune your pipelines!  
- Understand all of the nitty gritty details of how we structure all of the data in a Cell Painting dataset by reading [Data Structure in the Cell Painting Gallery](https://broadinstitute.github.io/cellpainting-gallery/data_structure.html).

## I’ve done image analysis and extracted image features. How do I make profiles?

- Read the blog post [How to Normalize Cell Painting Data](https://carpenter-singh-lab.broadinstitute.org/blog/how-normalize-cell-painting-data)  
- For a big picture overview, read the paper [Data-analysis strategies for image-based cell profiling](https://www.nature.com/articles/nmeth.4397)  
- Read about the metric we use for quantifying phenotypic strength in [A versatile information retrieval framework for evaluating profile strength and similarity](http://doi.org/10.1038/s41467-025-60306-2)

## I have profiles made from Cell Painting data. How do I explore biology in them?

- Watch the videos [The beginner's guide to morphological profiling (Morphological profiling, part 1)](https://youtu.be/0nkYDeekhtQ) and  [The beginner's guide to morphological profiling (Morphological profiling, part 2)](https://www.youtube.com/watch?v=r9mN6MsxUb0)  
- Read the blog post [Interpreting Image Based Profiles](https://carpenter-singh-lab.broadinstitute.org/blog/help-interpreting-image-based-profiles)  
- Follow the protocol [Interpreting Image-based Profiles using Similarity Clustering and Single-Cell Visualization](https://currentprotocols.onlinelibrary.wiley.com/doi/full/10.1002/cpz1.713).

## How do I understand Cell Painting features?

- Start by watching Ask Erin Dear Beth Episode 25 - [Interpreting Cell Painting Features: Classic Cell Profiler Features](https://youtu.be/13KDkqLhDRQ) for a casual podcast explanation.  
- Read [What do CellPainting features mean](https://github.com/carpenterlab/2022_Cimini_NatureProtocols/wiki/What-do-Cell-Painting-features-mean%3F) in the Cell Painting wiki  
- Read [How measurements are named](https://cellprofiler-manual.s3.amazonaws.com/CellProfiler-4.2.1/help/output_measurements.html) in the CellProfiler manual  
- For more specific details on any specific feature/measurement, read the [Definition of all measurements](https://cellprofiler-manual.s3.amazonaws.com/CellProfiler-4.2.1/modules/measurement.html) in the CellProfiler manual.

## How do I find published Cell Painting data to explore?

- The Cell Painting Gallery hosts a LOT of published Cell Painting data and is free to access. Read [Cell Painting Gallery Documentation](https://broadinstitute.github.io/cellpainting-gallery/overview.html).  
- [JUMP Hub](https://broadinstitute.github.io/jump_hub/howto/interactive/0_overview.html) has a set of introductory and advanced data exploration tools for JUMP, a single large Cell Painting dataset.
